Connect to GDP Presto programmatically

Skip to end of metadata
  • Created by Manhar Verunnukunnel Hassan, last modified on May 02, 2017
Go to start of metadata

Instructions to connect to gdp presto query engine using various languages.

Note: Before get started, please make sure you have active LDAP account. If not, please send a request to servicedesk@grubhub.com

Option 1: Setup your local machine for presto connectivity

  • Setup ssh tunnel for gdp environment as follows
    1. Connect to VPN (Split connection)
    2. Open your terminal window on Mac/Linux or command window on PC

    3. Copy and paste either of the following commands based on which environment you would like to connect

      Important Note

      These commands will never give you a shell. They will merely run in memory, exposing the port for use on your local machine. 

      1. DEV:    ssh -o ServerAliveInterval=10 -N -L 8889:localhost:8889 <ldap user>@dev-presto.gdp.data.grubhub.com
      2. PROD: ssh -o ServerAliveInterval=10 -N -L 8889:localhost:8889 <ldap user>@presto.gdp.data.grubhub.com
    4. Enter your LDAP password and hit enter. If you see any errors, please contact teamdatabase@grubhub.com

 

  • Sample code to connect to presto using pyhive once the above bridge is set up

    Sample python code

    from pyhive import presto
      
    prestoconnector = presto.connect('localhost', 8889)
    cursor = prestoconnector.cursor()
    cursor.execute('select * from hive.source_events.order_taking_events limit 10')
    print cursor.fetchall()
      

     

     

Option 2: Connect using gdp ssh util

     Please contact jboylan@grubhub.com or teamdatabase@grubhub.com in case of any issues.

  • Connect to VPN (Split connection)
  • Open your terminal window on Mac/Linux or command window on PC
  • git clone https://<gituser>@stash.grubhub.com/scm/data/sshutils.git
  • cd sshutils
  • python setup.py install

 

  • Sample code to connect to presto using sshutils

    Sample python code
    ?
    1
    2
    3
    4
    5
    6
    7
    8
    9
    import sshutils
    import getpass
    import pandas as pd
    password = getpass.getpass()
    prestoconnector = sshutils.get_presto_connector('dev-presto.gdp.data.grubhub.com', 'jboylan', password)
    query = 'select * from hive.source_events.order_taking_events limit 10'
    data = pd.read_sql(query, prestoconnector)
    print data
      

Option 3: Pyspark on Presto

  • ssh to presto 
    • Dev: ssh dev-presto.gdp.data.grubhub.com -o "StrictHostKeyChecking no"
    • Prod: ssh presto.gdp.data.grubhub.com -o "StrictHostKeyChecking no"
  • Open Pyspark
